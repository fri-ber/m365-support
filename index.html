Microsoft 365

Anleitungen
📧 Shared Mailbox einrichten

Exchange Admin Center → Recipients → Mailboxes → + Add shared mailbox. Berechtigungen unter «Mailbox delegation» vergeben (Full Access / Send As).

👥 Gruppe in Teams erstellen

Teams → Teams-Tab → «Join or create a team» → «Create team». Typ wählen (Privat/Öffentlich), Mitglieder hinzufügen, Kanäle strukturieren.

🔐 MFA aktivieren

Entra Admin Center → Users → Per-user MFA oder Conditional Access Policy «Require MFA» für alle oder bestimmte Gruppen.

📁 SharePoint Berechtigungen

Site Settings → Site Permissions → Gruppen anpassen. Vererbung bei Bedarf aufheben («Stop inheriting permissions»).

📊 Lizenz zuweisen

Entra Admin Center → Users → Lizenz auswählen → Apps und Dienste bei Bedarf deaktivieren. Bulk-Zuweisung via PowerShell empfohlen ab 10+ Usern.

🔄 Postfach-Migration

Exchange Admin Center → Migration → + Add migration batch. Typ «Remote move» oder «Cutover» je nach Szenario. Completion manuell triggern.

PowerShell Scripts

Kopieren & Ausführen
💡
Vor dem Ausführen: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser und Module installieren mit Install-Module ExchangeOnlineManagement
AD-Passwort-Ablauf-Check.ps1
# Zeigt alle User deren Passwort in X Tagen abläuft
$DaysUntilExpiry = 14

Get-ADUser -Filter * -Properties PasswordLastSet, PasswordNeverExpires |
  Where-Object { -not $_.PasswordNeverExpires } |
  Select-Object Name, SamAccountName,
    @{ Name='ExpiresOn'; Expression={ $_.PasswordLastSet.AddDays(90) } } |
  Where-Object { $_.ExpiresOn -lt (Get-Date).AddDays($DaysUntilExpiry) } |
  Sort-Object ExpiresOn |
  Format-Table -AutoSize
M365-Lizenzen-Report.ps1
# Verbindung zu M365
Connect-MgGraph -Scopes "User.Read.All", "Directory.Read.All"

# Alle lizenzierten User exportieren
Get-MgUser -All -Filter "assignedLicenses/`$count ne 0" `
  -ConsistencyLevel eventual -CountVariable $count |
  Select-Object DisplayName, UserPrincipalName, AccountEnabled |
  Export-Csv "./Lizenzen-Report.csv" -NoTypeInformation -Encoding UTF8

Write-Host "✅ Export fertig: Lizenzen-Report.csv" -ForegroundColor Green
Shared-Mailbox-Berechtigungen.ps1
# Shared Mailbox Zugriff vergeben
Connect-ExchangeOnline

$Mailbox = "support@firma.ch"
$User    = "hans.muster@firma.ch"

# Full Access
Add-MailboxPermission -Identity $Mailbox `
  -User $User -AccessRights FullAccess -AutoMapping $true

# Send As
Add-RecipientPermission -Identity $Mailbox `
  -Trustee $User -AccessRights SendAs -Confirm:$false

Write-Host "✅ Berechtigungen gesetzt für $User"
Deaktivierter-User-Cleanup.ps1
# Alle deaktivierten AD-User anzeigen und in OU verschieben
$ZielOU = "OU=Deaktiviert,DC=firma,DC=ch"

Get-ADUser -Filter { Enabled -eq $false } -Properties LastLogonDate |
  Where-Object { $_.DistinguishedName -notlike "*Deaktiviert*" } |
  ForEach-Object {
    Move-ADObject -Identity $_.DistinguishedName -TargetPath $ZielOU
    Write-Host "Verschoben: $($_.Name)"
  }
